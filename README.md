# x

## 介绍

**Node.js** 和 **MongoDB** 开发的社区系统。

## 安装部署

[Node.js](https://nodejs.org) v8.9.1，[MongoDB](https://www.mongodb.org) v3.4.9。

```
1. 安装 `Node.js[必须]` `MongoDB[必须]`
2. 启动 MongoDB
3. `npm install` 安装依赖包
4. 将 config.default.js 改为 config.js 请根据需要修改配置文件
5. `node app.js`
6. done!
```
